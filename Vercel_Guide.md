# Vercel 배포 시스템 및 대시보드 활용 가이드

## 1. 내 Vercel URL
- **대시보드 접속 주소**: [https://vercel.com/yjkim12345s-projects](https://vercel.com/yjkim12345s-projects)

## 2. Vercel이 하는 일 (역할)
Vercel은 '더와이즈 로펌 홈페이지'를 전 세계 인터넷망에 24시간 365일 안정적이고 가장 빠르게 송출해 주는 **최첨단 클라우드 호스팅 플랫폼**입니다.
- **Next.js 특화 렌더링**: 로펌 홈페이지를 만든 기술 단위(Next.js)의 창시 회사가 제공하는 플랫폼으로 최적화된 성능 제공
- **자동화된 배포(CI/CD)**: 코드가 변경되면 별도의 복잡한 서버 업로드 절차 없이 자동으로 수정된 홈페이지 버전을 배포
- **서버리스 함수(Serverless Functions)**: 문의 폼 등 백엔드 기능이 필요할 때 데이터베이스(Supabase)나 알림 봇(n8n)에 중간 다리 역할을 해주는 보안 서버 제공

## 3. 오늘 한 일 (작업 내역)
- **초기 시스템 배포**: 로컬 PC(`thewiselaw-homepage` 폴더)에만 있던 홈페이지 소스 코드를 Vercel 클라우드 공간에 업로드하여, 인터넷 브라우저로 접속할 수 있는 'Live' 상태의 실서버 구축
- **환경 변수 세팅 설계**: 홈페이지가 여러 데이터베이스 및 외부 서비스와 소통하기 위해 보안이 유지되는 `.env.local` 시크릿 키 파일을 설계 및 설정 준비

## 4. 향후 활용 용도
1. **무중단 실시간 업데이트**: 향후 새로운 승소 사례 코너를 만들거나, 변호사 프로필, 칼럼, 소개 글 디자인 등을 변경한 후 깃허브(GitHub) 저장소에 코드를 밀어넣기만(Push) 하면, Vercel이 이를 감지해 자동으로 최신 버전 홈페이지로 갈아 끼워줍니다.
2. **초고속 글로벌 CDN 네트워킹**: 엣지 네트워크(Edge Network) 기술을 통해 한국은 물론 전 세계 주요 노드에서 병목 없이 홈페이지 로딩 스피드를 최상단으로 끌어올립니다.
3. **SEO(검색엔진 최적화) 부스팅 탑재**: 빠른 로딩 속도와 서버 사이드 렌더링(SSR) 기술 덕분에 추후 구글, 네이버 검색 노출에서 훌륭한 가산점을 받을 수 있습니다.

## 5. 자주 들어가셔야 하는 이유 (즐겨찾기 필수)
이곳은 홈페이지 운영 중 반드시 정기적으로 방문하셔야 하는 **메인 관제탑(Control Tower)** 역할을 합니다.

- **[필수] 도메인 연결 설정 (Domains 메뉴)**: 향후 `thewiselaw.co.kr` 이라는 가장 폼 나는 공식 대표 도메인을 구매하신 후 런칭할 때, 이 대시보드 안에서 DNS 연결을 설정해야 합니다.
- **[필수] 환경 변수 값 갱신 (Environment Variables 메뉴)**: 로펌 홈페이지 문의 폼 연동 시 구축될 Supabase 키, n8n 웹훅 주소 등이 추가되거나 변경되면 반드시 Vercel 설정에서 값을 업데이트해 주셔야 기능이 먹통되지 않습니다.
- **배포 상황(Logs) 모니터링 (Deployments 메뉴)**: 코드 수정본을 올렸을 때 코드가 잘 적용이 되었는지(초록색 Ready 상태) 아니면 에러가 났는지(빨간색 Error 상태)를 한눈에 확인할 수 있습니다.
